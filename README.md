# 🎮 Tic Tac Toe

A simple Tic Tac Toe game built with **React**.  
Players take turns placing **X** and **O** on a 3x3 grid until one player wins or the game ends in a draw.  

---

## ✨ Features
- Two-player gameplay (X vs O).
- Detects win conditions (rows, columns, diagonals).
- Displays the winner.
- Reset button to restart the game.

---

## 🛠️ Tech Stack
- **React.js**
- **JavaScript (ES6)**
- **CSS (custom styling)**
